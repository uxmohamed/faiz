# Faiz — Arabic UX Copy Writer

An agent skill that writes Arabic UI copy for the Thmanyah (ثمانية) sports streaming platform.

## What it does

Generates consistent Arabic microcopy — buttons, errors, loading states, validation messages, empty states, confirmation dialogs, onboarding prompts — in a semiformal Saudi-conversational voice.

## Install

Clone into your Cursor skills directory:

```bash
git clone https://github.com/uxmohamed/faiz.git ~/.cursor/skills/faiz
```

Or add it to your project for the whole team:

```bash
git clone https://github.com/uxmohamed/faiz.git .cursor/skills/faiz
```

## Usage

Ask Cursor to write Arabic UI copy and the skill activates automatically. Examples:

- "Write an error message for a failed payment"
- "What should the loading state say while processing the subscription?"
- "Write the confirmation dialog for deleting a card"
